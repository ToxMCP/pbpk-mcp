# PBPK MCP v0.2.1

Suggested GitHub release title:

- `PBPK MCP v0.2.1: documentation, release-surface, and architecture cleanup`

Suggested short release summary:

- PBPK MCP `v0.2.1` consolidates the public documentation around the dual-backend MCP design, clarifies `rxode2` as a first-class execution engine, and cleans up the GitHub-facing README and architecture diagrams.

## Highlights

- repaired and simplified the public Mermaid architecture diagram
- aligned the README with the current accessibility plus OECD-oriented positioning
- clarified direct `rxode2` support for native R-authored PBPK models
- cleaned up release-facing documentation and metadata surfaces

## Limitations To Call Out Publicly

- Berkeley Madonna `.mmd` remains a conversion source, not a runtime backend
- `.pksim5` remains an export source rather than a direct runtime format
- qualification state and runtime support are separate concerns
- many included example profiles are still illustrative or research-oriented

## Suggested Release Notes Body

```markdown
PBPK MCP `v0.2.1` is a documentation and release-surface cleanup over `v0.2.0`.

This patch release:

- improves the GitHub-facing README structure
- repairs the Mermaid architecture diagram
- clarifies `rxode2` as a first-class PBPK execution backend
- aligns the public project description with the current dual-backend MCP design

No major runtime behavior changes are introduced in this patch release.
```
